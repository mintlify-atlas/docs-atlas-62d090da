## Find the best repos for anything

RepoHunt is a browser-based tool that cuts through GitHub noise. Instead of scrolling through hundreds of irrelevant results, RepoHunt uses multiple quality signals to rank repositories and generates AI-powered summaries so you can instantly understand what each project does — and whether it's worth your time.

## Why RepoHunt

GitHub search is powerful but shows you what's popular — not necessarily what's best for your use case. RepoHunt combines eight ranking signals and AI summarization to surface repositories that are high-quality, actively maintained, and relevant to your query.

## Get started

<Steps>
  <Step title="Get a free Groq API key">
    Visit [console.groq.com](https://console.groq.com) and create a free account to get your API key.
  </Step>
  <Step title="Open RepoHunt">
    Clone the repository and open `repohunt2.html` in your browser — no build step required.
  </Step>
  <Step title="Enter your key">
    Paste your Groq key into the prompt on first load. It's stored only in your browser's `localStorage`.
  </Step>
  <Step title="Search for anything">
    Type a query like `trading bot`, `AI agent framework`, or `Python web scraper` and hit **Search**.
  </Step>
</Steps>

<Note>
  RepoHunt uses the GitHub Search API (unauthenticated) and Groq API for AI summaries. No server, no database — everything runs client-side.
</Note>
